# 角色卡编辑器 创建由 OoCTalk 

>基于[Next.js](https://nextjs.org/)框架的现代化响应式角色卡编辑器，适用于创作SillyTavern等JSON格式的AI角色卡的创建、编辑、导入、导出。

[在线演示](https://ce.ooctalk.com)

## 特性 

- 无需保存
- 黑夜模式
- 多语言支持
- 更好的性能

## 开始部署

```bash
git clone https://github.com/ooctalk/CharacterEditor.git
cd CharacterEditor
npm install
npm run build
npm run dev & npm run start
```
